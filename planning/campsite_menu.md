# San Rafael Swell - Campsite Selection Menu

Bucketed by the end-of-day arrival window for each camping night. Candidates are ordered by mile; on-track (<= 250 m) called out.

Legend:
- `(mi)` = miles along route
- `off m` = distance from nearest point on main track (0-50 m = on the track; 50-250 m = short spur; >250 m = noticeable detour)
- "numbered" = part of a BLM-designated site cluster (Buckhorn Draw, Wedge Overlook, Temple Mountain) - each site typically fits 2-3 vehicles.
